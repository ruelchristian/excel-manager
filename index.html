Sideloading Setup Guide

To run custom HTML/JS side panels in Microsoft Excel Cloud (or Desktop), you must sideload the add-in. Since it requires a secure secure https:// URL, the add-in is hosted right here on GitHub Pages.

  • 1

    Prepare your manifest.xml

    Ensure your local manifest.xml points to this live GitHub URL on line 18:

    <SourceLocation DefaultValue="https://ruelchristian.github.io/excel-manager/taskpane.html"/>
  • 2

    Open Excel Cloud

    Log into OneDrive in your browser and open your master workbook: ADB MONTHLY (MASTER FILE) FINAL.xlsx.

  • 3

    Upload the Manifest

    Go to the top ribbon menu, click the Insert tab, select Add-ins (or Office Add-ins), click Upload My Add-in in the top-right corner, and select your edited manifest.xml.

  • 4

    Start Managing

    The ADB Subscription Manager sidebar will slide open on the right side of your worksheet.

  • 1

    Create a Shared Network Folder

    Create a folder on your local computer (e.g. C:\ExcelAddins) and place your manifest.xml inside it.

  • 2

    Share the Folder & Configure Trust

    Right-click the folder > Properties > Sharing > click Share (add yourself). Copy the Network Path (e.g. \\YOUR-PC\ExcelAddins).

  • 3

    Register in Excel Trust Center

    Open Excel Desktop > File > Options > Trust Center > Trust Center Settings > Trusted Add-in Catalogs. Paste the network path, click Add Catalog, and check the Show in Menu box.

  • 4

    Insert the Add-in

    Restart Excel. Open your sheet, go to Insert > My Add-ins > Shared Folder, select ADB Subscription Manager, and click Add.

Features Overview

The ADB Subscription Manager was built to replicate your original Google Sheets dashboard while taking advantage of Excel's speed, offering date protections and automated pastel coloring.

Real-Time Search

Type keywords to instantly filter records by End User (EU), Sales Officer (SO), Product, or Subscription ID across thousands of rows.

Date Range Filters

Filter active subscriptions by start or end date thresholds using an integrated popup calendar (flatpickr).

Add & Edit Forms

Modify records via sidebar forms. Click any list item card to pull its data, edit fields safely, and save it back to the sheet.

Data Protections

Prevents Excel from auto-converting remaining/total months (like 3/12) into dates by locking number formats as Text.

Multi-Tier Sorting

Orders your rows intelligently: active items are brought to the top (New -> Renewal -> Complete -> Cancelled), then sorted by nearest expiration date.

Row Highlight Colors

Applies custom, professional pastel colors across entire rows in Excel to help you distinguish renewal statuses in a split second.

Usage Guide & Walkthrough

1. Searching and Filtering

When you open the sidebar, it automatically displays your subscription cards. You can:

  • Keyword Search: Use the search bar to locate specific End Users, SOs, Products, or Subscription IDs.
  • Status Filter: Select a status tab (All, Active/New, Renewal, Complete, Cancelled) to narrow down items.
  • Date Range Filter: Click the date inputs to filter subscriptions active during a specific period.

2. Adding a New Subscription

Instead of scrolling to the bottom of the worksheet and manually typing values:

  1. Navigate to the Add/Edit Form tab.
  2. Fill in all the required fields (End User, Product, Month, Start & End Dates, etc.).
  3. Click the Save Subscription button. The entry will be written to Excel, and number formatting is applied automatically to secure inputs.

3. Editing & Deleting Entries

  1. Find the entry in the list view (under the Monthly or Annual Master tabs).
  2. Click the card. The sidebar will automatically switch to the form tab and populate all fields.
  3. Make your edits, then click Save Subscription.
  4. To remove it, click the red Delete button at the bottom of the form, and confirm when prompted. The row will be deleted, shifting rows up.

4. Generating Monthly Master Template Sheets

If you want to create a clean template sheet for a specific month:

  1. Under the Monthly Master tab, click the green Generate Monthly Sheet button at the top.
  2. Select the desired target month (e.g. January) from the dropdown.
  3. Click Generate Template. A new template worksheet will be initialized and pre-formatted.

Sorting Algorithm

When you click Re-Sort & Format Sheet, the system executes a 3-tier sorting algorithm directly inside your Excel worksheet:

  1. Tier 1: Status Importance - Brings active subscriptions (New/Active first, then Renewal) to the very top. Complete/Closed rows are pushed below them, and Cancelled rows are moved to the bottom.
  2. Tier 2: Start Date (Descending) - Within each status group, items are ordered by their start dates, placing the most recent subscriptions at the top.
  3. Tier 3: End User Name (Alphabetical) - If statuses and start dates match, rows are organized alphabetically by their End User (EU) name.

Troubleshooting & FAQ

Q: Why are sorting and formatting not happening automatically when I open the panel?

A: To prevent automatic, unwanted modifications to your workbook on start, auto-sorting at startup has been disabled. If you want to sort and highlight all rows, scroll to the bottom of the list view and click the Re-Sort & Format Sheet button manually.

Q: What is the "Legacy sheet layout detected" warning?

A: This warning appears if the Monthly Master sheet has an older layout structure (9-column or 10-column layout) instead of the current 12-column layout. When this is detected, the add-in stops loading to prevent data misalignment. Clicking the Re-Sort & Format Sheet button will automatically convert, re-align, and format the sheet into the correct 12-column layout.

Q: Excel converted my remaining months (e.g. 1/12) into a date. How do I fix this?

A: Excel tends to auto-detect numeric fractions as dates. To prevent this, the add-in pre-formats data columns as Text (@) before writing. If you have existing cells that converted, click the card in the sidebar to open it in the form, verify the remaining/total months value (e.g., 1/12), and click Save. The add-in will write it back as text and fix the cellular display.

Q: Can I run this in Excel Desktop without internet access?

A: The sideloaded add-in requires access to the hosted files (taskpane.html and taskpane.js) on GitHub Pages, so an active internet connection is required to fetch and run the sidebar scripts.
